# Guía de Funciones de Excel

En esta guía encontrarás enlaces a la documentación oficial de algunas funciones de Excel que pueden servirte como apoyo para la resolución de los ejercicios de la tarea.

## Aviso importante

Las funciones incluidas en esta guía no constituyen la única forma de resolver los ejercicios propuestos. En muchos casos, un mismo problema puede resolverse utilizando distintas funciones o combinaciones de ellas.

Asimismo, esta lista no es exhaustiva y pueden existir otras funciones que también resulten útiles según el enfoque adoptado. Se recomienda consultar la documentación oficial de Microsoft Excel para ampliar la información y explorar alternativas.

---


## COUNTA
**Descripción:** Devuelve la cantidad de celdas no vacías dentro de un rango. Considera números, texto, valores lógicos y errores.  
- 📄 **Documentación oficial:** [COUNTA - Microsoft](https://support.microsoft.com/en-us/office/counta-function-7dc98875-d5c1-46f1-9a82-53f3219e2509)

---

## COUNTIFS
**Descripción:** Devuelve la cantidad de celdas que cumplen múltiples criterios.
- 📄 **Documentación oficial:** [COUNTIFS - Microsoft](https://support.microsoft.com/en-us/office/countifs-function-dda3dc6e-f74e-4aee-88bc-aa8c2a866842)

---


## SUM
**Descripción:** Devuelve la suma de un conjunto de números o rangos de celdas.  
- 📄 **Documentación oficial:** [SUM - Microsoft](https://support.microsoft.com/en-us/office/sum-function-043e1c7d-7726-4e80-8f32-07b23e057f89)

## SUMIF

## VLOOKUP (BUSCARV)

- 📄 **Documentación oficial:** [VLOOKUP - Microsoft](https://support.microsoft.com/en-us/office/vlookup-function-0bbc8083-26fe-4963-8ab8-93a18ad188a1)
